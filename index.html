ShipSync by Andrew Holland

ShipSync automatically syncs your Etsy orders to your Veeqo account in real time. It ensures that every new Etsy sale updates the correct shipping information in your Veeqo order — without manual input.

Key Features

  • Real-time Etsy order detection via Etsy API
  • Matches and updates the correct Veeqo order using the Etsy receipt number
  • Shipping address sync via secure OAuth2
  • No email parsing — 100% API driven

About the Developer
